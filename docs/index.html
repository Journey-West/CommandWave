[ FEATURES ]

Terminal Management

Create and manage multiple terminal sessions with tmux and ttyd. Rename, switch, and synchronize your workflow.

Playbook Integration

Upload Markdown files as playbooks with executable code blocks. Build interconnected knowledge bases.

Variable Substitution

Create custom variables for your code. All variables are synchronized in real-time across sessions.

Cyberpunk Themes

Choose from Cyberpunk Dark, Neon Light, and Witch Hazel themes to personalize your experience.
