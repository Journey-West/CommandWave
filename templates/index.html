Key Features

Terminal Management
Create, switch between, rename, and close multiple terminal sessions backed by tmux and ttyd.
Playbook Integration
Upload Markdown files as playbooks, parsed to text/code blocks. Enable cross-references for an interconnected knowledge base.
Full Synchronization
All application data synchronized across clients.
  • Terminal tabs sync between sessions
  • Playbooks sync in real-time
  • Variables sync across browser windows and sessions
  • Persistent data between sessions
  • Consistent variable values across all connected clients
Variable Substitution
Define and use variables across your playbooks and terminal sessions.
  • User-friendly variable definition
  • Automatic substitution in code blocks
  • Variables persist and synchronize across sessions
  • Create custom named variables
  • Real-time synchronization of variable values between clients
Command Execution
Execute commands directly from playbooks to the active terminal.
